# 비로그인 정책 랜딩페이지 QA 테스트 (Landing Page QA)

## 테스트 개요

**목적**: 비로그인 정책 랜딩페이지의 모든 기능이 정상 작동하는지 검증

**범위**:
- 컴포넌트 렌더링 (UI)
- 인증 상태별 동적 UI
- 하이퍼링크 및 라우팅
- 반응형 디자인
- 로그아웃 기능
- 성능 (로딩 속도)

**테스트 환경**:
- Next.js 15.5.6
- Node.js 18+
- 브라우저: Chrome, Firefox, Safari (크로스 브라우저)
- 기기: 모바일 (375px), 태블릿 (768px), 데스크톱 (1440px)

**테스트 일자**: 2024-11-10
